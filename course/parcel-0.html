                【parcel】基本使用

                太阳丶 2018-7-12 parcel

                # 安装parcel

                npm install -g parcel-bundler

                # 新建一个项目

                创建index.html和main.js。

                <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<body>
<h1>hello world</h1>
<script src="./main.js"></script>
</body>
</html>

                main.js 是 parcel的入口文件。

                运行parcel

                parcel index.html

                服务会默认运行在localhost:1234上，如果1234端口被占用，则会随机使用一个空闲的端口

                Server running at http://localhost:64067 - configured port 1234 could not be used.
√  Built in 44ms.

                # 资源

                Parcel 支持 CommonJS 和 ES6 模块语法来导入文件。

                // 使用CommonJS语法导入一个模块
const a = require('./js/a');

// 使用ES6模块语法导入一个模块
import a from './js/a';

// 导入一个CSS文件
import './css/index.css';

// 通过css模块导入一个css文件
import base from './css/index.css';

// 通过 URL 导入一张图片
import image form './img/logo.png';

                # less

                安装 less 模块就可以使用less预编译了。

                npm install less

                # sass

                SCSS编译需要 node-sass 模块。可以用 npm 来安装它：

                npm install node-sass

                一旦 node-sass 安装完成，你就可以在 JavaScript 文件中导入 SCSS 文件。SCSS 文件中的依赖可以使用 @import 语句。

                import './custom.scss'

                # PostCss

                在parcel-bundler中已经集成了PostCss 模块。在根文件夹下创建postcss.config.js文件。

                module.exports = {
  plugins: [
    require('autoprefixer')({ browsers:  [
                              'last 20 versions',
                              'IE 9',
                              'iOS >= 8']})]
}

                # babel

                安装babel插件

                npm install babel-cli --save

npm install babel-preset-env --save

                新建.bablerc 文件，配置

                {
    "presets": [
        "env"
    ]
}
